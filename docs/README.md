# tunnel-tailscale — docs

Tailscale Funnel driver for the togo `tunnel` subsystem.

- **Install:** `togo install togo-framework/tunnel-tailscale`
- **Source:** https://github.com/togo-framework/tunnel-tailscale
- **Base:** https://github.com/togo-framework/tunnel

Requires the `tailscale` CLI + Funnel enabled. Set `TUNNEL_DRIVER=tailscale`.
See the [README](../README.md).
